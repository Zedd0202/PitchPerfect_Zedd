**kick**
--------

> - 볼륨을 슬라이더를 통해 조절합니다. 그냥 바로 녹음된 파일의 볼륨을 올리면 적용이 되지 않기 때문에, 버퍼를 이용해 녹음 파일을 버퍼안에 넣고 그 버퍼안의 볼륨을 조정하는 방식으로 구현.
